Основные консольные команды Git

Ссылка на гит-репозиторий

Настройка Git

  • git config --global user.name "Ваше имя" - Установка имени пользователя
  • git config --global user.email "ваш@email.com" - Установка email пользователя
  • git config --global --list - Просмотр текущих настроек

Создание и клонирование репозитория

Команда Описание
git init Инициализация нового репозитория
git clone "url-репозитория" Клонирование удалённого репозитория

Основные команды работы с файлами

Команда Описание
git add . Добавить все изменения
git commit -m "Сообщение" Создать коммит
git status Показать состояние репозитория

Работа с ветками

Команда Описание
git branch Показать список веток
git checkout -b имя_ветки Создать и переключиться на новую ветку
git merge имя_ветки Слить ветку с текущей
